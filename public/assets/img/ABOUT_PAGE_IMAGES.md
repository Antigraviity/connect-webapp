# About Page Images Guide

## Required Images for About Page

Please add the following images to make the About page complete:

### 1. Know About Us Section

- **Path**: `/public/assets/img/about-team.jpg`
- **Description**: A professional team photo showing 3-4 people working together
- **Dimensions**: Recommended 800x500px
- **Style**: Professional, bright, collaborative atmosphere

### 2. Start As Seller Section

- **Path**: `/public/assets/img/seller-working.jpg`
- **Description**: Image of a service provider or seller at work (cleaning, repair, etc.)
- **Dimensions**: Recommended 800x500px
- **Style**: Action shot showing professional service delivery

### 3. Trusted By Section (Company Logos)

- **Path**: `/public/assets/img/partners/`
- **Files needed**:
  - `partner-1.png` - Company/Business logo 1
  - `partner-2.png` - Company/Business logo 2
  - `partner-3.png` - Company/Business logo 3
  - `partner-4.png` - Company/Business logo 4
  - `partner-5.png` - Company/Business logo 5
- **Dimensions**: Recommended 200x80px (maintain aspect ratio)
- **Style**: Transparent background PNG format preferred

## Temporary Solution

The page will work without these images but will show:

- Generic placeholder images with text
- The layout and functionality will remain intact

## Note

All images should be optimized for web:

- Format: JPG for photos, PNG for logos
- Quality: 80-85% compression for JPGs
- Max file size: 200KB per image recommended
- Use tools like TinyPNG or ImageOptim for compression

## Color Scheme Reference

The page uses Company name's color scheme:

- Primary Blue: #0053b0
- Secondary Yellow: #fdd201
- Maintain consistency with existing brand guidelines
